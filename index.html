Ubuntu Firewall Management



Installation

The Ubuntu Firewall can be installed by first updating your repo by :

apt-get update && apt-get upgrade

Install the package
apt install ufw

Checking If Firewall Is Active

The firewall status can be checked by:

sysemctl status ufw

Starting Firewall

Starting the Firewall deamon can be done by:

ufw start

Staring The deamon allowing connections

ufw default allow outgoing

Making Rules

Allowing A service can be done by:
ufw allow socket or protocol

ufw allow ssh
or ufw allow 22
Allowing A specific Ip to connect to A specific port

ufw allow from (your Ip) to any port 22 proto tcp

Viewing Rules

The existing Rules Can Be seen By:


ufw status numbered

Deleting Rules

Rules Can be deleted by:
ufw delete (rule number)

ufw delete 1
